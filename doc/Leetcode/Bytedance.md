##### 402 removeKdigits

题目描述： 给你一个以字符串表示的非负整数 `num` 和一个整数 `k` ，移除这个数中的 `k` 位数字，使得剩下的数字最小。请你以字符串形式返回这个最小的数字。

```
num = 1432219
k = 3
res:	1219


num: 10200
k = 1
res : 200
```

数字比较: 对于两个相同长度的数字序列，最左边不同的数字决定了这两个数字的大小

删除一个数字的贪心策略：

- [D0D1D2 ... Dn-1] 从左往右找到第一个位置 i (i > 0) 使得Di < Di-1 => 删除Di-1
- 如果不存在Di < Di-1 说明序列递增，删除最后一个字符
