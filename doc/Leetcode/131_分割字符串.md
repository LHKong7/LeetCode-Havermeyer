给你一个字符串 s，请你将 s 分割成一些子串，使每个子串都是 回文串 。返回 s 所有可能的分割方案。

回文串是正着读和反着读都一样的字符串。



```
输入：s = "aab"
输出：[["a","a","b"],["aa","b"]]

输入：s = "a"
输出：[["a"]] 
```



整体思路如下：

（1）子字符串的截取方式

最短的可能性为1，最长可能是整个字符串

（2）回文的判断

可以翻转字符串，和原来字符串相等，则认为是回文。

也可以，从前往后进行比较（有点双指针的感觉），遇到不一致的则认为不是。

（3）终止条件

当截取的 index 等于源字符串的长度-1
